# 操作系统课程设计基础部分

本项目按照课程设计“基础必做部分”拆分为四个可独立完成的子项目，每个子项目单独放在一个文件夹中，便于分阶段开发、测试、演示和撰写报告。

## 目录说明

- `01_处理机调度`：实现 FCFS、SJF、RR、优先级调度等算法
- `02_内存管理`：实现页面置换与动态分区管理算法
- `03_进程同步与并发控制`：实现经典同步问题
- `04_文件系统`：实现简易文件系统或目录管理模块
- `scripts`：统一启动、测试和结果汇总脚本
- `output`：统一输出目录，保存测试结果和运行日志

## 开发建议

建议按以下顺序逐个完成：

1. 处理机调度
2. 内存管理
3. 进程同步与并发控制
4. 文件系统

每个子项目都建议包含以下内容：

- `README.md`：中文说明文档
- `src`：源代码
- `tests`：测试输入或测试脚本
- `output`：本模块输出结果

## 统一脚本

根目录提供统一脚本：

- `scripts/run_all.ps1`：检查目录结构、尝试编译并运行各模块、汇总输出结果

运行方式：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\run_all.ps1
```

如果本机未安装 C 编译器，脚本会给出提示信息，但目录检查和结果汇总仍可正常执行。
